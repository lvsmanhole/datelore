Design System

The Almanac — DateLore visual system

A warm, printed-almanac mood crossed with a birthday card and a modern timeline: aged parchment, espresso ink, amber gold, and a coral pop for celebrations. The whole system is plain HTML and CSS custom properties — one webfont (Newsreader for display and big numbers) plus the system sans — built to clear WCAG AA contrast and to stay ad-ready without ever letting the ads dominate.

Body — system sans at --step-0

The body text is set in the platform's native sans-serif at --step-0, with a 1.6 line height for comfortable reading. Newsreader is reserved for headings and big numbers so the single webfont stays cheap. On May 31, the almanac remembers the launch of the Titanic, the Johnstown Flood, and the first tick of Big Ben.

Mono caption — --font-mono at --step--2

Eyebrows · tags · captions · the 151st day of the year

Type scale is a fluid clamp() ramp from --step--2 (≈0.75rem) to --step-6 (≈6.3rem), tuned mobile-first between 360px and 1240px.
